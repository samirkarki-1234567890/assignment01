# assignment01
mobile development first assignement
----------------------------------------------------------
STEP 1: Download node.js from https://nodejs.org/en/download/ and install it by selecting everything to default

STEP 2: check if node.js is installed correctly by running the following command in the terminal:
        node -v
        npm -v
note: if not installed, try git bash or powershell

STEP 3: Once installed, open VS code and create a new folder for workspace

STEP 4: Then open the terminal in VS code(ctrl+`) and run the following command to create a new expo project:
        npx create-expo-app@latest IncredibleTodoListApp
        NOTE: [use git bash if other command line is not working]

STEP 5: 

STEP 6: Follow the instructions to install the app 'expo go' on your phone 

STEP 7: Run the app using the following command:
        npx expo start

STEP 8: Scan the QR code from the terminal with the expo go app on your phone
